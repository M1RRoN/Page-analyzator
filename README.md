# Page Analyzer

Web-service for check status of websites. [Live Demo](https://seo-page-analyzer-production-8868.up.railway.app/)


## Features

* Check status code
* Parse text from title tag
* Parse text from h1
* Parse meta-description


## Install

```powershell
make install
```

## Usage

```shell
make start
```
